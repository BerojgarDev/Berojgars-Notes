Web technology

  • Web Technology refers to the various tools and techniques that help to communicate between the different types of devices over the network by using the internet.
  • It consists of two words: web and technology.
  • The term web simply refers to World Wide Web(www), is the cyber space containing webpages, documents and other resources which can be accessed by using their URL.
  • The term Technology refers to the tools and techniques that make resources available on the web such as web browser, programming language and databases and protocols.

Web Technology can be classified into the following:

  1. World Wide Web(WWW)

    • It is also known as a Web, is a collection of websites or web pages stored in web servers and connected to local computers through the internet. The three basic languages that make up www are HTML, CSS and JavaScript.

  2. Web pages

    • A webpage is the collection of the digital document (text, images, videos, hyperlinks) that is linked to the WWW and viewable by the user. These web pages can be accessed through the different types of web browsers.

  3. Web Browsers

    • The web browser is a software to explore the Web. It provides an interface between the server and the client. So that the users can send requests to the server for web documents and services.
    • Example: Chrome, Microsoft Edge, Safari etc.

  4. Web Server

    • Are the software or hardware systems that store, process and deliver web pages to the users.
    • They respond to requests from the web browser and give responses to the users through the browser.
    • Examples:- Apache, Nginx, Microsoft Internet Information Service(IIs) etc.

Web Development

  • Web development involves the process of building, creating, and maintaining websites or web applications. It includes different tasks such as web design, web publishing, web programming, and database management, etc.
  • In one word, it is the creation of an application that works over the internet i.e. websites.

Classification of Web Development

  1. Front-End Development

    • Front-End Development is also known as client-side development. It is the process of developing the visual elements of the website that are visible and interactive to the end users.
    • The person who is responsible for development of client-side or front-end is called as Front-End Developer
    • It includes the design, layout, and user interface of the website.
    • Technologies Used: HTML, CSS, JavaScript

  2. Back-End Development

    • Back-End Development is also known as server-side development. It is the process of developing the server-side logic, databases, and APIs that power the website.
    • A person who is responsible for development of the backend side of the web is called as Backend Developer
    • It includes the task related with database interactions, user authentication, server, network and hosting configuration, and business logic.
    • Technologies Used: PHP, Python, Node.js, Ruby, Java, MySQL, PostgreSQL

  3. Full-Stack Development

    • Full-Stack Development is the process of developing both the front-end and back-end of a website.
    • A person who is responsible for development of the full-stack side of the web is called as Full-Stack Developer
    • Technologies Used: HTML, CSS, JavaScript, PHP, Python, Node.js, Ruby, Java, MySQL, PostgreSQL

Web development languages

  1. Front-End languages

    1. HTML
      • HTML stands for Hypertext Markup Language.
      • It is used to design the front-end portion of web pages using a markup language.
      • HTML is the combination of Hypertext and Markup language. Hypertext defines the link between the web pages. The markup language is used to define the text documentation within the tag which defines the structure of web pages.
    2. CSS
      • Cascading Style Sheets simply referred to as CSS is a designed language intended to simplify the process of making web pages presentable.
      • CSS allows you to apply styles to web pages.
      • More importantly, CSS enables you to do this independent of the HTML that makes up each web page.
    3. JavaScript
      • JavaScript is a famous scripting language used to create magic on the sites to make the site interactive for the user.
      • It is used for the functionality of a website to run cool games and web-based software.
      • It also provides different frontend frameworks like React JS, Angular, Vue JS,Next.JS etc

  2. Back-End languages

    1. PHP
      • PHP is a server-side scripting language designed specifically for web development.
      • Since, PHP code is executed on the server-side, it is called a server-side scripting language.
    2. Python
      • Python is a high-level, interpreted programming language known for its simplicity and readability.
      • It is widely used in web development, data science, artificial intelligence, and more due to its extensive standard library and community support.
    3. Node.js
      • Node.js is a JavaScript runtime built on the V8 engine that allows you to run server-side JavaScript code.
      • It enables event-driven, non-blocking I/O operations, making it highly scalable for building fast web servers and APIs.
    4. Ruby
      • Ruby is a dynamic, interpreted programming language known for its elegant syntax and focus on developer productivity.
      • It is widely used for web development, particularly with the Ruby on Rails framework, which emphasizes convention over configuration.
    5. Java
      • Java is a high-level, OOP language designed to be platform-independent.
      • It is widely used for building robust, secure, and scalable applications, including web, mobile, and enterprise systems.
    6. C#
      • It is a general-purpose, modern and OOP language pronounced as “C sharp”.
    7. DBMS
      1. MySQL
        • MySQL is a popular open-source relational database management system (RDBMS) that stores and retrieves data using structured query language (SQL).
        • It is widely used for web applications, offering reliability, scalability, and ease of use.
      2. PostgreSQL
        • PostgreSQL is a powerful, open-source relational database system known for its reliability, extensibility, and strict SQL compliance.
        • It supports complex queries, transactions, and various data types, making it suitable for enterprise-level applications.
      3. MongoDB
        • MongoDB is a popular NoSQL database that stores data in flexible, JSON-like documents.
        • It is widely used for web applications requiring scalability, flexibility, and high performance.
      4. SQLite
        • SQLite is a lightweight, self-contained, and serverless relational database engine.
        • Unlike client-server databases like PostgreSQL, SQLite is an embedded library that stores the entire database inside a single cross-platform file on your disk.

Methods of displaying output in JS

JavaScript Comment

  • Comments are a meaningful way to deliver messages, add information about the code, warnings or suggestions so that end users can easily interpret the code.
  • The JavaScript comment is ignored by the JavaScript engine i.e. embedded in the browser.

Types of JavaScript Comments

  1. Single Line Comment

    • It is represented by double forward slashes (//). It can be used before and after the statement.
    • Used to comment a single line of code.
    <script>
    // This is a single line comment
    alert("Hello World");
</script>

  2. Multi Line Comment

    • It can be used to add single as well as multi line comments.
    • It is represented by forward slash with asterisk then asterisk with forward slash.
    <script>
    /* This is a multi line comment
       It can span multiple lines
       and can be used to comment multiple lines of code */
    alert("Hello World");
</script>

Variable in JS

  • A JavaScript variable is simply a name given to storage location or we can also define variables as containers for storing data of different datatypes.

  • There are some rules while declaring a JavaScript variable.

    1. The name must start with a letter (a-z or A-Z), underscore or dollar ($) sign.
    2. After the first letter we can use digits (0 to 9), for example value1.
    3. JavaScript variables are case sensitive, meaning a and A are different.

JavaScript variable declaration and initialization

  1. Using var keyword:- var x, var first_name, var x = 10; var name=“Hari”;
  2. Using let keyword such as:- let _value; let x; let x=10;let name; etc
  3. Using const keywords such as:- const x; const pie; const price; const x=10; const pie=3.24; const price=5000; etc.

JavaScript variable scope

  • The scope of a variable is the region of your program in which it is defined.
  • There are two types of variable scope.

  1. Global Scope

    • A JavaScript global variable is accessible from any function. A variable i.e. declared outside the function or declared with a window object is known as a global variable.
    • Example: 
      <script>
    var x = 10; // Global variable
    function myFunction() {
        console.log(x); // Can access global variable
    }
    myFunction();
</script>

  2. Local Scope

    • A JavaScript local variable is declared inside a block or function and It is accessible within the function or block only.
    • Example: 
      <script>
    var y = 20; // Global variable
    function myFunction() {
        var z = 30; // Local variable
        console.log(y); // Can access global variable
        console.log(z); // Can access local variable
    }
    myFunction();
    console.log(y); // Can access global variable
    console.log(z); // Cannot access local variable
</script>

JavaScript Data Types

  • Data types are used to determine what type of data is stored in the variable.
  • There are two types of data types in JavaScript.

  1. Primitive Data Types

    • Primitive data types serve as the fundamental building blocks from which all other data types are constructed.

      DataType Description Example
      String Represents sequence of characters. name = ”hari”;
      Number Represents nubmeric value (both integer and floating point numbers). age=10;
      Boolean Represents true or false value. isStudent=true;
      undefined Represents a variable that has not been assigned a value. name;
      Null Represents an intentional absence of any object or value. name=null;

  2. Non-Primitive Data Types

    • Non-primitive data types which are also known as composite or reference data types.
    • These non-primitive data types allow to group and organize multiple values into a single variable.

    • The non-primitive data types in JS are as follows:

      DataType Description
      Object A collection of key value pairs. It represents an instance through which we can cross members of an object.
      Array It is a linear data structure that stores a collection of elements of the same data type.

    • Example of object:

      <script>
    let person = {  name: "Alice",
                age: 30,
                isStudent: false,
                email: "Alice33@gmail.com"
            };
</script>

    • In JS, we can access members of an object by using dot operator( ‘.’ ) or square bracket ( ‘[]’ ). Suppose if you want to access members of the above object.

      <script>
// Using Dot Notation
console.log(person.name); //Alice
console.log(person.email); //Alice33@gmail.com
console.log(person.name, person.age); //Alice 30
// Using Square Bracket Notation
console.log(person["name"]); //Alice
Console.log(person["age"]); //30
</script>

    • Example of Array:

      <script>
let fruits = ['Apple', 'Banana', 'Orange', 'Mango'];
// Accessing array members using index
console.log(fruits[0]); // Output: Apple
console.log(fruits[3]); // Output: Mango
// Modifying array members using index
fruits[1] = 'Grapes';
console.log(fruits); // ['Apple', 'Grapes', 'Orange', 'Mango']
// Finding the length of the array
console.log(fruits.length); // Output: 4
// Adding a new element to the end of the array
fruits.push('Strawberry');
console.log(fruits); // ['Apple', 'Grapes', 'Orange', 'Mango', 'Strawberry']
// Removing the last element from the array
fruits.pop();
console.log(fruits); // ['Apple', 'Grapes', 'Orange', 'Mango']
</script>

JavaScript Operators

  • JavaScript operators are symbols that are used to perform operations on operands.
  • For example:- var sum = 10 + 20;
  • Here, + is the arithmetic operator and = is the assignment operator.

  • There are following types of operators in JavaScript.

    1. Arithmetic Operators

      • Arithmetic operators are used to perform arithmetic operations on the operands.

      • The following operators are known as JavaScript arithmetic operators.

        Operator Description Example
        + Addition x = 10, y = 20, result = x + y; // result will be 30
        - Subtraction x = 10, y = 20, result = x - y; // result will be -10
        * Multiplication x = 10, y = 20, result = x * y; // result will be 200
        / Division x = 10, y = 20, result = x / y; // result will be 0.5
        % Modulus x = 10, y = 20, result = x % y; // result will be 10
        ++ Increment x = 10, result = x++; // result will be 10
        -- Decrement x = 10, result = x--; // result will be 10

    2. Comparison Operators

      • The JavaScript comparison operators are used to compare the two operands.

      • The comparison operators are as follows:

        Operator Description Example
        == Equal to x = 10, y = 20, result = x == y; // result will be false
        === Identical to x = 10, y = "10", result = x === y; // result will be false
        != Not equal to x = 10, y = 20, result = x != y; // result will be true
        !== Not identical to x = 10, y = "10", result = x !== y; // result will be true
        > Greater than x = 10, y = 20, result = x > y; // result will be false
        < Less than x = 10, y = 20, result = x < y; // result will be true
        >= Greater than or equal to x = 10, y = 20, result = x >= y; // result will be false
        <= Less than or equal to x = 10, y = 20, result = x <= y; // result will be true

    3. Logical Operators

      • JavaScript is used to perform logical operations on values. They are essential for making decisions in code, combining multiple conditions, and controlling the flow of a program.

        Operator Description Example
        && Logical AND x = 10, y = 20, result = x && y; // result will be true
        || Logical OR x = 10, y = 20, result = x || y; // result will be true
        ! Logical NOT x = 10, y = 20, result = !x; // result will be false

    4. Assignment Operators

      • JavaScript assignment operators are used to assign values to variables. They are essential for initializing and updating variable values in code.

        Operator Description Example
        = Assignment x = 10;
        += Addition assignment x += 10; // same as x = x + 10
        -= Subtraction assignment x -= 10; // same as x = x - 10
        *= Multiplication assignment x *= 10; // same as x = x * 10
        /= Division assignment x /= 10; // same as x = x / 10
        %= Modulus assignment x %= 10; // same as x = x % 10

    5. Special Operators

      • A variety of special operators provide ways to work with data and manipulate values.

        Operator Description
        ?: The conditional operator returns value based on the condition. It is like if-else.
        , The comma operator evaluates each of its operands in order from left to right, and returns the value of the last operand.
        delete The delete operator removes a property from an object.
        in The in operator returns true if the specified property is in the specified object or its prototype chain.
        instanceof The instanceof operator returns true if the specified object is an instance of the specified constructor.
        new The new operator creates a new object.
        typeof The typeof operator returns the type of a variable.
        void The void operator evaluates the expression and returns undefined.

JavaScript Control Structure

  • Control structures are just a way to specify the flow of control in a program. It basically analyzes and chooses the direction.

    1. JavaScript If statement

      • It evaluates content only if the expression is true.

      • Example:

        <script>
    var a = 20;
    if (a > 10) {
        document.write("value of a is greater than 10");
    }
</script>

    2. JavaScript If else Statement

      • It evaluates the content whether the condition is true or false.

      • Here is an example:

        <script>
    var a = 10;
    if (a > 10) {
        document.write("value of a is greater than 10");
    } else {
        document.write("value of a is less than 10");
    }
</script>

    3. JavaScript Nested If Statement

      • It evaluates the content only if the expression is true from several expressions.

      • Example:

        <script>
var a = 20;
if (a == 10) {
    document.write("a is equal to 10");
}
else if (a == 15) {
    document.write("a is equal to 15");
}
else if (a == 20) {
    document.write("a is equal to 20");
}
else {
    document.write("a is not equal to 10,15,20");
}
</script>

    4. JavaScript Switch Statement

      • Switch statement is used to execute one code from multiple expressions.
      • It is just like the else if statement that we have learned in the previous page.

      • Example:

        <script>
var a = 10;
switch (a) {
    case 10:
        document.write("a is equal to 10");
        break;
    case 15:
        document.write("a is equal to 15");
        break;
    case 20:
        document.write("a is equal to 20");
        break;
    default:
        document.write("a is not equal to 10,15,20");
}
</script>

JavaScript Loops

  • The JavaScript loops are used to iterate the piece of code repeatedly until the certain condition is true.
  • It makes the code compact.
  • It is mostly used in arrays.

  • Types of loop:

    1. JavaScript For loop

      • The JavaScript for loop iterates the elements for the fixed number of times. It should be used if the number of iterations is known.

      • Syntax:

        <script>
for (initialization; condition; increment/decrement){
//statement to be executed
}
</script>

      • Example:

        <script>
for (var i = 0; i < 5; i++) {
    document.write(i + " ");
}
</script>

    2. JavaScript While Loop

      • Control flow statement that allows you to repeatedly execute a block of code as long as the specified condition is true.

      • Syntax:

        <script>
while (condition) {
    //statement to be executed
}
</script>

      • Example:

        <script>
while (i < 5) {
    document.write(i + " ");
    i++;
}
</script>

    3. JavaScript Do While Loop

      • It is similar to while loop but the body of the while loop is executed once then only the condition is checked.

      • Syntax:

        <script>
do {
    //statement to be executed
} while (condition);
</script>

      • Example:

        <script>
do {
    document.write(i + " ");
    i++;
} while (i < 5);
</script>

    4. “for in” Loop

      • The JavaScript for loop is used for iterating through the properties of an object.

      • Syntax:

        <script>
for (variable in object) {
    // code block to be executed
}
</script>

      • Example:

        <script>
 let person = {
       fname: "John",
       lname: "Doe",
       age: 25
   };
   for (let key in person) {
       document.write(key + ":" + person[key]);
   }
</script>

    5. “for of” Loop

      • The for of loop iterates over the values of an iterable object, such as an array or a string.

      • Syntax:

        <script>
for (variable of iterable) {
    // code block to be executed
}
</script>

      • Example:

        <script>
const arr = [1, 2, 3, 4, 5];
for (let value of arr) {
    console.log(value); // Outputs: 1 2 3 4 5.
}
</script>

Programming in JavaScript

  1. WAP in JS to take input from users and calculate their sum by using a function.

    function sum(a, b) {
        return a + b;
    }
    var num1 = prompt("Enter first number");
    var num2 = prompt("Enter second number");
    num1 = parseInt(num1);
    num2 = parseInt(num2);
    var result = sum(num1, num2);
    document.write("Sum of " + num1 + " and " + num2 + " is: " + result);
</script>

  2. WAP in to calculate the factorial of a number.

    <script>
    function factorial(){
        var n,i,fact=1;
        n = prompt("Enter a Number : ");
        n = parseInt(n);
        if (n < 0){
            alert("factorial not exist");
            return;
        }
        for(i = 1; i <= n; i++){
            fact = fact * i;
        }
        document.write("Factorial is "+fact);
    }
    factorial();
    </script>

  3. WAP to print Fibonacci series up to 10’th term.

    <script>
    let a = 0, b = 1;
    console.log(a);
    console.log(b);
    for (let i = 2; i < 10; i++) {
        const nextTerm = a + b;
        console.log(nextTerm);
        a = b;
        b = nextTerm;
    }
</script>

  4. WAP to find the greatest number among three.

    <script>
    var x,y,z;
    x = prompt("Enter first number");
    y = prompt("Enter second number");
    z = prompt("Enter third number");
    x = parseInt(x);
    y = parseInt(y);
    z = parseInt(z);

    if (x > y && x > z){
        document.write("Greatest number is: " + x);
    }
    else if (y > x && y > z){
        document.write("Greatest number is: " + y);
    }
    else{
        document.write("Greatest number is: " + z);
    }
</script>

  5. WAP to check whether the number is even or odd.

    <script>
    var n;
    n = prompt("Enter a number");
    n = parseInt(n);
    if (n % 2 == 0){
        document.write("The number is even");
    }
    else{
        document.write("The number is odd");
    }
</script>

  6. WAP to check palindrome or not?

    <script>
    var n, rev = 0, rem;
    n = prompt("Enter a Number : ");
    n = parseInt(n);
    while (n != 0)
    {
        rem = n % 10;
        rev = rev * 10 + rem;
        n = parseInt(n / 10);
    }
    document.write("Reversed number is: " + rev);
</script>

  7. WAP in JS to check palindrome using function.

    <script>
    function isPalindrome(str) {
        const reversedStr = str.split('').reverse().join('');
        return str === reversedStr;
    }
    var n = prompt("Enter a string");
    if (isPalindrome(n)) {
        document.write("The string is a palindrome");
    } else {
        document.write("The string is not a palindrome");

Introduction to PHP

  • PHP stands for Hypertext Preprocessor. It is a server-side scripting language designed for web development.
  • PHP is embedded in HTML code and interpreted by a web server with a PHP processor module, which generates dynamic web pages.
  • It is an open-source, cross platform scripting language and is widely used for web development purposes.
  • It performs different tasks such as handling forms, interacting with databases, and creating dynamic and interactive websites.
  • It is called server-side scripting language because, when a user makes a request to a web server, the server processes the request, executes the server-side script, interacts with databases if needed, and then sends the result (usually HTML, CSS, or other client-readable content) back to the client's browser.

Features/Advantages of PHP

  1. Open Source:- PHP is freely available and can be used, modified, and distributed without any cost
  2. Server-Side Scripting:- PHP is primarily designed for server-side scripting, allowing developers to create dynamic web pages by executing code on the server.
  3. Cross-Platform:- PHP is platform-independent, meaning it can run on various operating systems such as Windows, Linux, macOS etc.
  4. Integration:- PHP can easily integrate with various databases, including MySQL, PostgreSQL, SQLite, and more, making it well-suited for building database-driven web applications.
  5. Extensive Library Support:- PHP has a rich set of libraries and extensions that provide additional functionalities, simplifying common tasks such as handling sessions, working with XML, parsing JSON, and more.
  6. Community Support:- The PHP community is large and active, offering differing resources, forums, and documentation.

Weakness/Disadvantages of PHP

  1. PHP's main strength and flexibility is also its weakness. It can be a little too forgiving of errors.
  2. With no strict rules, inexperienced programmers have the freedom to create some very bad solutions to simple problems.
  3. PHP is an interpreted language, which can reduce speed.
  4. While PHP has improved its security features over time, improper use of the language can still lead to security vulnerabilities. like SQL injection, cross-site scripting (XSS), and other common web application security threats.
  5. Object-oriented patterns may not be as polished as in some other programming languages.

  6. Despite these weaknesses, PHP remains a popular and widely used language for web development, especially for building dynamic websites and web applications. Many successful websites and applications are built using PHP, and ongoing improvements are made to address its limitations.

GET and POST Methods in PHP

  • In PHP, GET and POST are HTTP request methods used to transfer data from a client (web browser) to a web server.
  • These methods are commonly used while submitting data through HTML forms. PHP provides superglobal arrays $_GET and $_POST to access the data sent by these methods.

  1. GET Method:-

    • The GET method sends data to the server by appending it to the URL in the form of name–value pairs.

    • Characteristics of GET method:

      1. Data is visible in the browser’s address bar
      2. Limited amount of data can be sent
      3. Less secure as data can be easily seen
      4. Data can be bookmarked and cached
      5. Mainly used to retrieve or search data
    • Example: $_GET['username'];

  2. POST Method:-

    • The POST method sends data to the server inside the HTTP request body, not through the URL.

    • Characteristics of POST method:

      1. Data is not visible in the URL
      2. Large amount of data can be transmitted
      3. More secure than GET
      4. Data cannot be bookmarked
      5. Suitable for sending sensitive data (passwords)
    • Example: $_POST['username'];
