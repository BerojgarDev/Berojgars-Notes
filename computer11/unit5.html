Some Terminology in Web Technology

Web Browser

  • It is an application software for accessing the information on the world wide web (WWW). It helps to insert, update and retrieve information such as text, images, animations, audio, etc. from the client side to the server side and vice-versa.
  • A client side device might be a PC, workstation, tablet or smartphone and server side devices areweb server, mail server, data centers, etc.
  • A user can request information from the website and the web browser retrieves the information from the respective web server and presents it in the web browser.
  • Most popular web browsers are Google’s Chrome, Microsoft’s Edge, Apple’s Safari, Opera-mini, fireFox, etc.

Search Engines

  • A search engine is a web application or website which helps to search the information from the world wide web. The searched information contains webpages, links, articles, etc.
  • Mainly a search engine contains three components. They are: web crawler, indexing, and Searching.
  • The search engines retrieve the information by crawling from one page to another. The crawler is also known as spider. The indexing system makes an index of a file according to the information provided by the spider, searching algorithm helps to get the best result by using indexing files.
  • The most common examples of search engines are Google, Yahoo, Bing, Duck Duck Go, etc.

Internet

  • It is defined as the network of networks that consists of millions of inuliduals computers, mobile phones, academic, government and non-government organisations to communicate and share information between them.
  • It consists of two major components: protocol and hardware.
  • The term protocol simply refers to the set of rules. The Internet uses the Transmission Control Protocol or Internet Protocol(TCP/IP).
  • The internet was originally developed by DoD(Development of Defence) USA in 1969 by scientist Robert
  • Taylor and Lawrence Robert for the communication between different government agencies, military, etc.

WebPage

  • Webpage is the collection of electronic documents provided by the website to the users on the internet. We can post our information in the form of text, pictures, audio, videos, etc. as webpages.
  • A webpage is developed by using HTML and we can traverse from one page to another page using special links which are called hyper links.

Website

  • A collection of different webpages is called a website.

Types of Website

Static Website

  • The website that has simple design and has no interactive features is called a static website. These websites are used to deliver the information to the users exactly as stored and are developed by using HTML and CSS only.
  • These websites contain text, graphic, audio, video, etc. such websites periodically updated by manually editing.

  • Example:

    portfolio websites, resume website, documentation sites.

Dynamic website

  • Dynamic website is the interactive website with capability to change the content each time they are accessed.
  • The dynamic website is controlled by the server where it is stored, it can be easily updated and developed by using the programming language like JavaScript, PHP, Java, ASP, etc.

  • Example:

    YouTube, Netflix, Facebook, etc.

Web-Development

  • Web development refers to the process of creating and maintaining such websites or web applications.
  • It involves various tasks such as designing, coding, testing, deploying, etc.


Web-Technology

  • The term web technology refers to the collection of tools, protocol, programming language and different frameworks that are used in the development and maintenance of websites and web applications.

  • The key components of it are as follows.

    1. HTML:- Hypertext MarkUp Language is the backbone of web content HTML structure the element on a webpage.
    2. CSS:- Cascading Style Sheet is used for styling the HTML elements. It controls the presentation, layout and design of the website.
    3. JavaScript:- A dynamic scripting language that enables client side interactively and manipulation of the webpage content.
    4. HTTP/HTTPS:- Protocols that are used for transferring data between a web server and user’s browser.
    5. WebServer:- The different software like Apache, NGINX, Microsoft Information Service, etc. that host and save the web content.
    6. Backend Development:- It involves server side programming by using a different language like PHP, Python, NodeJS, etc. which handles data processing the database interaction.
    7. Database:- The different database management systems like MySQL, MongoDB, PostgreSQL, etc. are used to store & manage data for web applications for websites.

Content Management System (CMS)

  • A content management system often called CMS is a software application that helps the user to create, manage and modify the content on a website without requiring technical expertise.
  • It is a tool that is helpful to develop the website without writing all the code from scratch. It provides different templates to create and manage different digital contents on a website.

  • It consists of two parts:-

    1. Content Management Application (CMA):- It is a frontend part of software i.e. helpful to users to add or modify the content of a website.
    2. Content Delivery Application (CDA):- It is a backend part of software i.e. helpful to store the content visible on the website.


Introduction to HTML

  • HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages.
  • Unlike programming languages that are used to write instructions for the computer to follow and perform tasks, HTML is markup language that defines the structure & layout of a webpage using various tags and series of elements.
  • Father of HTML:- Tim Bemer Lee in 1990 (European lab Switzerland).
  • Different versions have been released and the current version is HTML-5.

Basic HTML page

<!DOCTYPE html>
<html>

<head>
    <title>My First Page</title>
</head>
<body>
    <p> Hello World!!</p>
</body>

</html>

Case Sensitivity of HTML

  • HTML is not case sensitive, this means you can write your HTML tags and attributes using uppercase, lowercase, or a mixture of both and the browser will interpret them the same way.
  • For eg: <ul> and <ul> are treated as the same tag.
    <html>    =   <HTML>
    <p>       =   <P>
    <head>    =   <Head>
    <body>    =   <BODY>

Comments in HTML

  • The comments are a meaningful way to deliver messages or add information about the code.

  • Comments are always ignored while executing the codes.

CSS

  • CSS stands for Cascading Style Sheets. It describes how HTML elements are to be displayed on the screen and control the layout of multiple webpages all at once. CSS is used to define style for your webpages, including the design layout and variations in display for different devices and screen sizes.

Types of CSS

  • CSS can be ulided into 3 types, they are as follows.

1. Inline CSS

  • If CSS is applied directly to an HTML element using the style attribute then it is called inline CSS.
  • For eg:
<h1 style="color: red; font-size: 16px;">This is an Example</h1>

2. Internal CSS

  • If CSS is defined within the <style></style> element in the <head> section of an HTML document then it is called internal CSS.
  • For eg:
<html>

<head>
    <style>
        h1{    
            color: blue;    
            font-size: 16px;
        }
    </style>
</head>

<body>
    <h1>Hello world</h1>
</body>

</html>

3. External CSS

  • If CSS is defined in a separate file and linked to an HTML document using the <link> element then such CSS is called external CSS.
  • For eg:

Inside index.html

<html>    

<head>        
    <link rel="stylesheet" href="style.css">    
</head>    

<body>        
    <h1 id="hi">Hello world</h1>    
</body>

</html>

Inside style.css

    #hi{        
        color: red;    
    }
