Program

  • A program is a sequence or set of instructions written to perform a specific task or set of tasks by a computer or machine.
  • Programs are usually written in programming languages (such as C, Python, C++, or Java) that a computer can interpret and execute.

Programming

  • The process or activity of writing program is called programming, which makes works easier and faster.

Programming Language

  • A programming language is a set of rules and symbols that are used to telling a computer to carry out a certain task.
  • It serves as a bridge between humans and machines, enabling us to create software, applications, and websites.
  • Programming languages are also known as computer language.

Features of good Programming Language

  • A good programming language should exhibit several key features here are some important features:-
    1. Simplicity and Readability:- The syntax should be straightforward and easy to understand, making the code easier to read and maintain.
    2. Efficiency:- The language should allow for efficient execution, using system resources like memory and CPU effectively.
    3. Portability:- The language should be platform-independent, allowing programs written in it to be executed on different systems.
    4. Modularity:- The language should support breaking down a large system into smaller, manageable, and reusable components.
    5. Security:- The software should protect sensitive data and prevent unauthorized access.

Types of Programming Language

  • Computer languages can be categorized into two types, each serving different purposes and having unique characteristics. Here are the main types:

    1. LOW LEVEL LANGUAGE

      • A low-level language is a type of programming language that is very close to the computer's hardware it means that the language directly interacts with the computer's physical components like the CPU and memory.
      • Low level language are also called machine dependent language which means program written for one type of system cannot be run in another system.
      • Further low-level programming languages are divided into two types:-

        1. Machine Level Language[1GL]:-

          • Machine level language which is also known as machine code is the lowest-level programming language. It consists of binary digits (0 and 1) that the computer's CPU can execute directly.

          • This language is specific to each computer's architecture and is used to perform basic operations like arithmetic, data movement and control instructions.

            Advantages(Merits)
            1. It is directly understood by the processor so has faster execution.
            2. Language translator is not needed.
            3. Requires less memory space as compared to high level programming languages.
            Disadvantages(Demerits)
            1. It is very difficult to program using 0 and 1s.
            2. Use of this language makes programming time consuming.
            3. Difficult to find errors and to debug.

        2. Assembly Language[2GL]

          • It is also known as low level programming language. This language uses symbolic instructions in place of 0 and 1, which are called mnemonics.
          • Examples includes ADD, SUB, JMP,RET, PRINT etc.

          • The program written in assembly language is converted into machine code by assembler thus resulting program is called an object code.

            Advantages of Assembly level language
            1. Assembly language allows for highly optimized code that can run faster than code written in higher-level languages.
            2. Programs written in assembly language can be very compact, making efficient use of memory
            3. Since assembly language is closer to machine code, it doesn't rely on compilers to optimize the code.
            Disadvantages of Assembly level language
            1. Writing program in assembly language is complex and time consuming.
            2. Debugging and maintaining assembly code can be challenging.
            3. Assembly language is specific to a particular computer architecture.

    2. High Level Language

      • A high-level language is a type of programming language designed to be more user-friendly and closer to natural language, making it easier for humans to read and write code.
      • Unlike low-level languages that are closely aligned with a computer's hardware, high-level languages use syntax and semantics that resemble human languages.

      • It uses familiar English words, special characters and mathematical symbols in their syntax which makes easier to read, write and understand.

        Advantages of High-Level Language
        1. Programmer friendly and easy to understand.
        2. Portable and machine independent.
        3. Easy to modify and maintain.
        4. Easy to debug.
        Dis-advantages of High-Level Language
        1. Need to be translation into machine code.
        2. Slower execution time.
        3. Less control over hardware.
        4. High memory usage.

      • High Level Language is also divided into two types.

        1. Procedural Oriented Programming Language(POP) or 3GL
          • It is also high-level programming language which is primarily focus on procedure or steps.
          • It follows top to bottom approach.
          • Examples C, FORTRAN,COBOL etc.
        2. Object Oriented Programming Language(OOP) or 4GL
          • This approach is based on the data rather than procedure (function and programs are normally divided into objects.
          • All the modern programming languages like Java, Python C++, dot net follow this approach.

Programming Language Translator

  • Translator which is also called language processor is a software tool that converts a computer program written in one language to another language.
  • It takes a program written in high level language which is called source code, and convert that source code into machine code which is directly understand by CPU.
  • It also helps to find out the error during translation process.

  • There are 3 different types of Translator

    1. Assembler

      • Assembly language is a low-level programming language that is closely related to machine code but is more readable for humans.
      • An assembler is a type of software that translates assembly language code into machine code, which is the binary code that a computer's processor can execute directly.
      • Assemblers can be categorized into two types based on the number of passes they take to process the input.

    2. Compiler

      • A compiler is a software tool that translates code written in a high-level programming language (such as C, C++, Java, or Python) into machine code or bytecode code which can be directly executed by a computer.
      • This translator reads the complete program at first if it is bug free then it converts the source code into machine code at once. If it detects the errors then the compilation process gets terminated. Thus, it also helps to find out the errors.
      • Simply compiler is processor dependent or platform dependent means the codes only works for the system, so, to overcome such problem cross compiler can be used.

    3. Interpreter

      • An interpreter is also a software program that translates a source code into a machine language.
      • However, an interpreter converts high-level programming language into machine language line-by-line while interpreting and running the program.
      • Interpreter reads only single statement of a program then translates it and execute it. After that it again read the next statement of the program, translate it and executes.

Errors

  • In programming errors are the mistakes or problems that makes while writing the codes, which prevents the execution of the program.
  • Errors are also called bugs and the process of removing the bugs is called debugging.
  • For successful execution of program errors must be removed.

  • Types of error

    1. Syntax error

      • Are those errors that mainly occurs due to the mistakes while typing a program or don't follow the syntax of program.

      • The most frequent syntax error are, missing parenthesis, semicolon. 

        #include <stdio.h>
int main(){
    printf("hello world") //missing semicolon
}

    2. Semantic error

      • Semantic error occurs when the statement written in the programs are not meaningful to the compiler.
      • Writing invalid program logic that produce incorrect result when the instructions are executed.
      • Suppose a programmer wants to add 2 numbers

      • Example

        sum = a - b;  // Error in semantics: wrong operator
sum = a + b;  // corrected version

    3. Runtime error

      • Runtime errors are those errors which are occur during program execution after successful compilation.
      • Example division by zero.

Control Structure

  • It is the way that specify the flow of program. It basically analyze and choose the direction flow of a program according to condition.

  • It can be categorized into 3 types.

    1. Sequential Structure

      • A sequential structure refers to the simplest control flow structure where statements are executed one after another. 

        <START>
    STATEMENT1;
    STATEMENT2;
    STATEMENT3;
<STOP>

    2. Selection Structure

      • Which is use to make decisions based on the several conditions.
      • It consists of two parameter: condition and statement

      • Types of Selection Structures in C:

        1. if statement

          • The simplest form of selection structure is the if statement. It evaluates a condition (which is a logical expression) and executes a block of code if the condition is true.

          • Syntax

            if (condition){
    // code to be execute if true
}

          • Example

            #include <stdio.h>
int main(){
    int num = 10;
    if (num > 0){
        printf("Number is positive\n");
    }
    return 0;
}

        2. if-else statement

          • The if-else statement provides two paths: one if the condition is true and another if the condition is false.

          • Syntax 

                if (condition) {
        // Block of code to execute if the condition is true
    } else {
        // Block of code to execute if the condition is false
    }

          • Example

                #include <stdio.h>
    int main(){
        int num = -5;
        if (num > 0){
            printf("Number is positive\n");
        }
        else{
            printf("Number is non-positive\n");
        }
        return 0;
    }

        3. else if ladder

          • The else if ladder is used when multiple conditions need to be checked. If the first condition is false, the next condition is evaluated, and so on.
          • If none of the conditions are true, an optional else block can be used as a default case.

          • Syntax

            if (condition1) {
    // code to execute
} else if (condition2) {
    // code to execute
} else if (condition3) {
    // code to execute
} else {
    // code to execute
}

          • Example

            #include <stdio.h>
int main() {
    int num = 0;
    if (num > 0){
        printf("Number is positive\n");
    }
    else if (num < 0) {
        printf("Number is negative\n");
    }
    else {
        printf("Number is zero\n");
    }
    return 0;
}

        4. nested if statement

          • A nested if is an if statement placed inside another if or else block. This allows for checking multiple conditions in a hierarchical manner.

          • Syntax

            if (condition1) {
    if (condition2) {
        // code to execute if both conditions are true
    }
}

          • Example

            #include <stdio.h>
int main() {
    int num = 5;
    if (num > 0) {
        if (num % 2 == 0) {
            printf("Number is positive and even\n");
        }
        else {
            printf("Number is positive and odd\n");
        }
    }
    return 0;
}

        5. switch statement

          • The selection control structure in which the case expression can be executed if the switch expression is true.

          • Here is syntax..

            switch (expression) {
    case constant1:
    //code to execute if expression equals constant1
    break;
    case constant2:
    //code to execute if expression equals constant2
    break;
    // You can have any number of case statements
    default:
    //code to execute if expression doesn't match any case
}

          • Example

        case 1:
        printf("Monday\n");

        case 2:
        printf("Tuesday\n");
        break;

        printf("Wednesday\n");
        break;
        case 4:
        printf("Thursday\n");
        break;
        case 5:
        printf("Friday\n");
        break;
        case 6:
        printf("Saturday\n");
        break;
        case 7:
        printf("Sunday\n");
        break;
        default:
        printf("Invalid day\n");
    }
    return 0;

    3. Iterative(Looping) Structure

      • In C programming, a looping structure (or iterative structure) allows you to execute a block of code repeatedivy until a certain condition is met, otherwise the program is terminated.

      • There are mainly two types of loops in C Programming:

        1. Entry Controlled loops

          • In Entry controlled loops the test condition is checked before entering the main body of the loop. For Loop and While Loop is Entry-controlled loops.

        2. Exit Controlled loops

          • In Exit controlled loops the test condition is evaluated at the end of the loop body. The loop body will execute at least once, irrespective of whether the condition is true or false. do-while Loop is Exit Controlled loop.

          • Types of Looping Structure in C

            1. For Loop
              • The looping structure in which the statement is executed at a specific number of times repeatedivy if the condition is true otherwise the program is terminated.
              • It belongs to the entry control loop.

              • Syntax

                for (initialization; condition; update) {
    // Block of code to execute
}

              • // Example of C program to illustrate for loop

                #include <stdio.h>
int main() {
    int i = 0;
    for (i = 1; i ≶ = 10; i++) {
        printf( "Hello World \n");
    }
    return 0;
}
            2. While Loop
              • The while loop in C programming is a control flow structure that allows you to execute a block of code repeatedivy as long as a specified condition remau true.
              • It belongs to the entry-controlled loop category, meaning that the condition is checked before the loop body is executed.

              • Syntax

                while (condition) {
    // block of code to execute
}

              • // C program to demonstrate while loop

                #include <stdio.h>
int main() {
    int i = 0;      // Initialization of loop variable
    while (i <5) {       // will execute till i is less than 5 
        printf(“Hello Class \n");       // loop statements
        i++;        // updating the loop variable
    }
    return 0;
}
            3. Do while loop
              • The do-while loop in C programming is an exit-controlled loop, meaning that the condition is evaluated after the loop body has been executed. This ensures that the loop body is executed at least once, even if the condition is initially false.

              • Syntax

                do {
    // code to execute
}
while (condition);

              • // C program to illustrate do-while loop

                #include ≶stdio.h>
int main() {
    int i = 2;      // Initialization expression
    do {
        printf( "Hello World \n");      // loop body
        i++;// Update expression
    }
    while (i <1);        // Test expression
    return 0;
}

C Programming Language

  • C is high-level general-purpose programming language. Which is developed by Dennis Ritchie in 1970 at AT&T Bell laboratory in USA.
  • This is procedure-oriented programming language that follow top-down approach. It mainly focus on procedure rather than objects like in other Programming Language.

  • It was invented to write an operating system called UNIX.

    Features of C Language

    1. C programs are portable i.e. they can be run on any compiler with little or without modification.
    2. C provides variety of data-types.
    3. It supports various control statements like if, if else and looping structures.
    4. It is more user friendly as compared to other Programming Language.
    5. It supports modular programming which means a complex program can be divided into different modules.

    Advantages of C programming Language

    1. Foundation of Modern Programming:C is the basic programming language and the mother of many modern languages like Python.
    2. Fast and Efficient:C compiles into compact, fast executable code.
    3. Library Support:Many libraries are written in C, making it highly versatile.
    4. Simple Syntax:C has minimal vocabulary, clear syntax, and is easy to learn.
    5. System-level Programming:C is close to assembly language, ideal for system-related tasks.
    6. Flexible and Logical:It allows clear program structure, meaningful variable names and flexible control statements.

    Disadvantages of C programming Language

    1. No Runtime Checking:Bugs are hard to find in large programs.
    2. Lacks Namespace:No way to avoid name conflicts.
    3. Weak type checking:Allows mismatched datatypes(Eg.: Integer passed as float).
    4. No Object-Oriented features:Lacks polymorphism, data hiding, constructors, and destructors.
    5. Limited Libraries:Insufficient for modern programming needs.
    6. Case Sensitivity:Mixing uppercase and lowercase can cause errors.
    7. Compatibility Issues:Difficult to run on modern 64bit systems.
    8. Not Designed-Oriented:Writing C code is not intuitive or easy.

    Array

    • It is the collection of similar data-type(Homogeneous) stored in contiguous memory location.
    • It can be declared as:
    • Syntax datatype arrayname[size];
    • Example int arr[5];
    • It can be initialized as int num[5]={1,2,3,4,5};
    • Float num[3]={1.2, 2.5, 2.2};

    • Array is used to store multiple values in a single variable instead of  declaring separate variable for each values.

      Advantages of Array

      1. Code Optimization:- Less code required to access data.
      2. Random Access:- We can access any element randomly using array.
      3. Simple syntax:- Array have simple and straight forward syntax.
      4. Easy Sorting:- We can sort the elements of array by using few line of code.

      Dis-advantages

      1. In C array have a fix size that need to be defined at the compile time.
      2. This makes it difficult to resize the array dynamically.

      Types of Array

      1. One-Dimensional Array( 1D Array)
        • In one dimensional Array the data are stored sequentially and have only one dimension.
        • Syntax datatype arrayname[size];
        • Int num[5];
      2. Multi-Dimensional Array
        • Are those array that have more than one dimension. Like 2D, 3D array.
        • int matrix[3][3];
        • here 3 is the number of rows and 3is the number of columns.

String

  • In 'C' programming string is typically represented as array of characters terminated by a null character '\0' .

  • Example

    char str[6] = "Hello";
// String declaration
  • Here, str is an array of 6 characters, where: 'H', 'e', 'l', 'l', 'o', '\0'

String Function

  • The header file <string.h> contain some string manipulation function. The most common string manipulation function in C are listed below.

    1. strlen()

      • The string function that returns the length of string.

      • For example, 

        #include <stdio.h>
#include <string.h>
int main() {
    char name[12];
    int length;
    printf(“Enter Name:- ”);
    scanf(“%s”, name);
    length=strlen(name);
    printf(“The length of the string is %d”, length);
    return 0;
}

    2. strcpy()

      • The string function that is use to copy a string from one location to another(from one variable to another).

      • For example, 

        #include <stdio.h>
#include <string.h>
int main() {
    char first[10];
    char second[10];
    printf(“Enter a string:”);
    scanf(“%s”, first);
    strcpy(second, first);
    printf(“The first string is %s and second string is %s”, first, second);
    return 0;
}

    3. strcmp()

      • This function is used to compare two strings and returns integer values indicating their relationship:

        1. Returns 0: If both strings are equal.
        2. Returns a negative value: If, first string<second string.
        3. Returns a positive value: first string > second string. 
        #include <stdio.h>
#include <string.h>
int main() {
    char str1[] = "Hello";
    char str2[] = "World";
    char str3[] = "Hello";
    int result1 = strcmp(str1, str2);
    int result2 = strcmp(str1, str3);
    printf("Comparison of str1 and str2: %d\n", result1);
    printf("Comparison of str1 and str3: %d\n", result2);
    return 0;
}

    4. strcat()

      • This function is use to join one string to another. It takes two string as arguments, the characters of the second string will be appended to the first string. This process of adding string is known as string concatenation.

        #include <stdio.h>
#include <string.h> {
    char str1[20] = "Hello";
    char str2[ 5] = " World!";
    strcat(str1, str2);
    printf("Concatenated String: %s\n", str1);
    return 0;
}

    5. strupr()

      • The strupr() function is a C library function used to convert all the characters in a string to uppercase. It stands for "string uppercase."

        #include <stdio.h>
#include <string.h>
int main() {
    char str[100];
    printf("Enter a string: ");
    scanf("%s", str);
    strupr(str);
    printf("Uppercase string: %s\n", str);
    return 0;
}

    6. strlwr()

      • This function is used to converts the uppercase characters of string to the lower-case character. 

        #include <stdio.h>

      • This function is used to reverse the given string.
