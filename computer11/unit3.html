Concept of Software

  • It is the collection of instruction, data and computer programs which are used to run machines and carry out the particular activities.
  • As we know the hardware cannot do anything itself, so to carry out the specific operation software is required.

Categories of Software

  • Software is simply categorized into two types. They are system software and application software.
  • They are further ulided into different types which are shown in figure below:

System software

  • The group of programs which allow the computer to communicate with different computer components or other hardware devices are called computer systems.
  • In simple words, system software is a collection of programmes that manage and support A computer Adelaide and application software.
  • It includes the operating system device drive and utility programme that helps the computer run smoothly.
  • It can be further device into following types.

Operating system

  • It is a system support that is stored on secondary devices such as HDD or SDD. When you power on your computer , the system firmware(BIOS) initiates the loading of as into RAM.
  • It provides a user interface enabling interaction with the system and running applications by allocating memory and processing power. Popular examples include windows,macOS and linux.Essentially, the OS is the bridge between users and users computers inner workings.

Package loader

  • A package loader is a component of an operating system responsible for loading and managing software packages or applications.
  • It locates the necessary files ,resolves dependencies ,and ensures the correct version of each package is loaded into memory. This process enables applications to run smoothly on the system.
  • The package loader acts as the mediators between the software and the operating system.

Language processor

  • A language processor translates high level programming code into machine code that the computer can execute. It includes assemblers, compilers and interpreters serving specific tasks.
  • Language processors ensure that code written by programmers is correctly transformed into inductions for the computer's hardware.

Utility software

  • Utility software is a program designed to help manage, maintain and optimize computer performance.
  • Examples include antivirus software, disk clean-up tools, firewall, etc. these tools assist in diagnosing issues, maintaining system health and improving efficiency.

Application Software

  • Application software includes programs designed to perform specific tasks for users, such as word processors, spreadsheets and web browsers. Those applications help users create documents, manage data, browse the internet, and user-centric tasks.
  • It can be further ulided into two types:

General Purpose(Package Software)

  • General purpose software is designed to perform a wide range of tasks, making it versatile and applicable to various user needs.
  • Examples: word processors, spreadsheet, browsers.
  • These programs can be used in many different contexts. Essentially, general-purpose software provides board functionality to support multiple activities.

Specific Purpose(Tailored Software)

  • Tailored Software, also known as specific purpose software, is specifically designed to perfectly meet the unique needs and requirements of an inulidual or organization.
  • If the users requirements are not met these software can change easily because these software are made by normal programmers or developers.
  • For Example: billing system, marks sheet generator, attendance & school management system, etc.

Operating System

  • It is an organised collection of software programs that control the overall resources of the computer system(Memory,CPU I/O devices).
  • It provides a platform for other software and applications to execute or run. It acts as an intermediate between user and the hardware which can be shown in figure below.
  • The most popular OS includes Linux OS, Mac OS, Windows OS, etc.
  • Today operating systems can be found in every device like smartphones, PCs, Auto Mobiles, T.Vs, etc.

Function of Operating System

  • The major functions of an OS are as follows:

  1. Memory Management

    • It refers to the management of the primary(main) memory.
    • During the execution of a program, it takes or shares a fixed amount of memory. After completion of execution it releases the memory. During the execution if the RAM is not sufficient to store the data the secondary memory is used which is called virtual memory.

  2. Processor Management

    • In a multiprogramming environment the OS decides which process gets the processor when and for how long, this process is called process schedule.
    • It allocates processors to execute processes and deallocates when a process no longer exists.

  3. I/O Management

    • An OS manages the different I/O devices by using their respective drivers. It decides which process gets the device when and for how long. It allocates I/O devices when required to perform specific operation and it deallocates the I/O devices when the process is finished.

  4. User Interface(UI)

    • It is the ideal environment in which the user can work on through which the user can interact with the computer.
    • Every OS provides the features of UI in order to enhance the experiences of computer systems. It acts as a bridge between user and computer.
    • There are various types of interfaces like GUI(Graphical User Interface), VUI(Voice User Interface), CLI(Command Line Interface), etc.

  5. Security

    • The OS contains a number of built-in tools to protect against viruses, security threats, unauthorised access and other suspicious activities.

  6. Deadlock Prevention

    • The deadlock is the situation where a set of processes are blocked because each process is holding a resource and waiting for another resource required by some other process.
  • So, the operating system helps to avoid the deadlock condition by proper resource allocation.

Process and its states

  • An instance of a program itself is not a process, it becomes a process when an executable file is loaded into the memory.
  • The different states of a process is shown in figure below.
  • New states:- In new states, the process being created.
  • Ready states:- The process is written into an assignment to a processor.
  • Running states:- In this state, the instructions are being executed.
  • Waiting states:- The process is waiting for some event to occur such as I/O completion, or reception of signals.
  • Terminate states:- In this state, the process is completed and process is terminated.

Types of Operating System

  • There are several types of OS, some of them are as follows.

1. Multi-programming OS

  • The OS in which more than one program can be executed at the same time is called a multi-programming OS. It is a technique in which multiple programs are executed simultaneously by a single processor.

2. Multi-tasking OS

  • Multi-tasking OS simply refers to the type of Multi-programming OS in which more than one task can be executed at a time.
  • In Multi-Tasking OS, there is only one CPU that switches between the different processes so quickly that gives the illusion that all processes are executing at the same time.

3. Multi-processing OS

  • A multi-processing OS uses two or more CPUs, to execute multiple processes simultaneously improving performance and reliability.
  • Each processor can run different processes or share tasks for faster execution. It is used in high-performance systems like services and super computers. This ensures better multi-tasking, fault tolerance and efficient CPU utilization.

4. Distributed OS

  • A distributed operating system is an operating system that runs on multiple computers but makes them work together like a single system.
  • It helps those computers share resources, work on tasks together, and appear as one to the user. It is designed to be more reliable and able to grow easily by adding more computers to the network.

Mobile Operating system

  • A mobile operating system is software that manages the hardware and software resources of a mobile device, like a smartphone or tablet.
  • It provides a platform for applications(apps) to run and enables user interaction with the mobile device.

Open-Source Operating System

  • Open-source generally refers to those OS whose source code is freely available on the internet. Basically the software that falls under general public licence is open source. They need not to buy the licence to use the software.
  • The open source OS code can be modified according to the users requirement and redistributed and redistributed to any other person. Some examples include UNIX, LINUX, SOLARIS, etc.

Linux Operating System

  • Linux is an open-source, Unix-like operating system that was first created by Linus Torvalds in 1991 at the university of Helsinki as a project and released the version of Linux including all the source code.
  • Unlike other operating systems, linux is free to use, modify and distribute, which has led to widespread use in various environments and a variety of software tools that provide a user-frienuly interface.
  • Anyone who uses the OS can also contribute to its improvement, whether it's fixing bugs, adding new features, or ensuring compatibility with new hardware. This has led to a global community of developers, enthusiasts, and companies who actively contribute to making Linux better and more widely used.

Advantages of Linux

  1. It is free of cost.
  2. There is no need to spend time and money to obtain a licence.
  3. It has higher scalability.
  4. It is flexible for high performance server applications, desktop applications and embedded systems.
  5. It is one of the most secure Operating Systems.

Unix Operating System

  • Unix is a powerful , flexible and manageable OS with good utilities and communication mobility. It is a multi-user, multi-tasking, and multi-processing operating system.
  • It is used in bigger size machines like super computers, mainframe computers and mini-computers.
  • It was developed by AT&T Bell laboratory around 1970 AD. Unix is developed in C programming language and development of Unix has given birth to many OS such as Linux, Solaris, etc. which inherits its main features from Unix.

Advantages of Unix

  1. Multi-User & Multi-Tasking:- UNIX supports multiple users and allows them to run multiple processes simultaneously without interference.
  2. Stability and Reliability:- UNIX is known for its ability to run continuously without crashing, making it highly reliable.
  3. Security:- UNIX has strong security features, including user authentication and file permission, to protect data and system integrity.
  4. Portability:- UNIX can be installed and run on different types of hardware, making it highly adaptable.
  5. Powerful Command-Line Interface:- UNIX provides a versatile command-line interface that offers powerful tools for system management and automation.
